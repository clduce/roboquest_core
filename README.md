# roboquest_core
Core ROS nodes for the RoboQuest Robot

## Installation

1. Create the Linux user ubuntu
2. Create the directory ~/catkin_ws/src
3. clone this repo into ~/catkin_ws/src
4. follow the procedure in docs/roboquest_update.txt
